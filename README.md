# Aqua

Aqua is a programming language for data stream analysis.

## Installation

Install [Rust](https://www.rust-lang.org/) and then:

```bash
cargo install aqua --git=https://github.com/aqua-language/aqua
```

## Usage

```bash
aqua
>> 1 + 1;
2
```
